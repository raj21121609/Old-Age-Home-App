# 🏡 Old Age Home Care App

A full-stack mobile application built using **Flutter (Frontend)** and **Node.js + Express + SQLite (Backend)** to manage elderly residents in an old age home.

---

## 📱 Features

### 👩‍⚕️ Caretaker Module
- Add and manage elderly residents
- View resident details
- Daily medication tracking
- Submit daily reports
- Alerts & notifications
- Profile management

### 🏛️ Government & Admin (Future Scope)
- Monitoring system
- Reports & analytics

---

## 🛠️ Tech Stack

- **Frontend:** Flutter
- **Backend:** Node.js + Express
- **Database:** SQLite
- **API:** REST APIs

---

## ⚙️ Setup Instructions

### 📌 1. Clone the Repository

```bash
git clone <your-repo-link>
cd Flutter_OldAgeApp

🔧 Backend Setup
📌 2. Go to backend folder
cd backend

📌 3. Install dependencies
npm install

📌 4. Start server
npx nodemon server.js

✔ Server will run on:
http://localhost:5000

📱 Frontend Setup (Flutter)

📌 5. Go to frontend folder
cd ../frontend

📌 6. Install dependencies
flutter pub get


▶️ Run the App
📱 Run on Android Device
flutter run

💻 Run on Chrome
flutter run -d chrome

⚠️ Important Notes
Make sure backend is running before frontend
Phone and laptop must be on same network
Do NOT use localhost for mobile testing
